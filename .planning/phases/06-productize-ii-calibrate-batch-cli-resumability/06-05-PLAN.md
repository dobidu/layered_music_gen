# Plan 06-05: Full CLI Rewrite

## Objective

Wave 4: replace the Phase 3 stub `src/musicgen/cli.py` with the full typer CLI implementing three commands — `generate`, `clean`, and `calibrate` — per D-61/D-62/D-63/D-64/D-65. Populate `tests/test_cli.py` with fast tests using `typer.testing.CliRunner`.

This plan depends on 06-04 (batch.py must exist for `generate` command to import).

## Tasks

### Task 1: Rewrite src/musicgen/cli.py

**File changes:**
- `src/musicgen/cli.py` (rewrite) — implement verbatim per `06-PATTERNS.md` Pattern 6.

Key implementation points:
1. Three commands: `generate`, `clean`, `calibrate`. All under one `app = typer.Typer(...)`.
2. `generate` command:
   - Required: `--seed S` (Ellipsis default → typer errors if missing).
   - Optional: `--count N` (default 1), `--out DIR` (default `"./dataset"`), `--workers W` (default None), `--output-mode M` (default `"full"`), `--verbose/-v`, `--quiet/-q`.
   - Validates `output_mode` against `_VALID_OUTPUT_MODES` before constructing Config (belt-and-suspenders — Config.__post_init__ also validates).
   - Calls `Config.load(cli_overrides={...})` then `generate_batch(cfg)`.
   - Exits with code 1 if `result.failed > 0`.
3. `clean` command:
   - Required: `--failed` flag (if not provided, prints usage hint and exits 1).
   - Optional: `--out DIR` (default `"./dataset"`).
   - Reads `manifest.jsonl` line-by-line; builds `{sample_index: last_status}` dict.
   - Removes dirs for `status == "failed"` AND no `sample.json` sentinel (D-64 last-status-wins).
   - Prints count of cleaned dirs.
4. `calibrate` command:
   - Optional: `--out DIR` (unused in logic; cache is always under `config.project_root`), `--verbose/-v`.
   - Calls `Config.load()` then `calibrate.measure_and_save_preroll(cfg.project_root)`.
   - Prints measured offset in seconds.
5. `_setup_logging(verbose, quiet)` helper — `logging.basicConfig(level=...)`.
6. `if __name__ == "__main__": app()` preserved at bottom.

No changes to `pyproject.toml` entry point (`musicgen = "musicgen.cli:app"` from Phase 3 Plan 03-01 — stays).

### Task 2: Populate tests/test_cli.py

**File changes:**
- `tests/test_cli.py` — remove `pytest.mark.skip`; implement 8 tests using `typer.testing.CliRunner`.

All tests use `CliRunner.invoke(app, [...])` which runs in-process (no FluidSynth invocation as long as `generate_batch` is mocked or the test avoids actual generation).

```python
from typer.testing import CliRunner
from musicgen.cli import app
runner = CliRunner()
```

Tests:
```
TestGenerateCommand:
  test_generate_missing_seed_exits_nonzero — invoke ["generate", "--count", "1"] (no --seed) → exit_code != 0
  test_generate_invalid_output_mode_exits_nonzero — invoke ["generate", "--seed", "42", "--output-mode", "invalid"] → exit_code != 0
  test_generate_help_exits_zero — invoke ["generate", "--help"] → exit_code == 0
  test_generate_runs_with_mocked_batch — monkeypatch generate_batch in cli module to return a BatchResult(total=1, succeeded=1, ...); invoke ["generate", "--count", "1", "--out", str(tmp_path), "--seed", "42"] → exit_code == 0

TestCleanCommand:
  test_clean_no_failed_flag_exits_nonzero — invoke ["clean"] → exit_code != 0
  test_clean_empty_manifest_exits_zero — invoke ["clean", "--failed", "--out", str(tmp_path)] (no manifest.jsonl) → exit_code == 0, output contains "nothing to clean"
  test_clean_removes_failed_dir — create manifest.jsonl with failed entry + dead dir (no sample.json); invoke clean --failed; dir removed; exit_code == 0

TestCalibrateCommand:
  test_calibrate_exits_zero — monkeypatch measure_and_save_preroll to return 0.0; invoke ["calibrate"] → exit_code == 0
```

For `test_generate_runs_with_mocked_batch`:
```python
def test_generate_runs_with_mocked_batch(tmp_path, monkeypatch):
    from musicgen.batch import BatchResult
    from musicgen.api import SampleResult

    fake_result = BatchResult(
        total=1, succeeded=1, failed=0, skipped=0,
        results=(SampleResult(
            sample_index=0, seed=0,
            sample_dir=str(tmp_path / "000000"),
            sample_json_path=str(tmp_path / "000000/sample.json"),
            mix_path="", stem_paths={}, midi_paths={},
            split="train", status="ok",
            musicality_score=0.5, duration_seconds=1.0,
        ),),
        duration_seconds=0.1,
    )
    monkeypatch.setattr("musicgen.cli.generate_batch", lambda cfg: fake_result)
    result = runner.invoke(app, ["generate", "--count", "1",
                                  "--out", str(tmp_path), "--seed", "42"])
    assert result.exit_code == 0
    assert "1 ok" in result.output
```
